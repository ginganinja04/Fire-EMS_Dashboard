Battalion Chief Dashboard

An interactive dashboard for visualizing fire-rescue incident activity, turnout performance, and battalion-level operational metrics.

Project Overview

This project is being developed for a graduate Information Visualization course at Florida Institute of Technology. The goal is to create an interactive dashboard that helps Battalion Chiefs monitor operational activity, compare unit performance, and identify meaningful trends in incident data.

Planned Dashboard Features

  • Turnout time compliance by unit
  • Incident volume and workload comparisons
  • Incident type breakdowns
  • Timeline-based incident activity views
  • Map-based incident visualization
  • Interactive filtering by date, shift, battalion, and unit

Development Status

The project is currently in early development. Initial work is focused on repository setup, data inspection, preprocessing pipeline design, and preparation of structured data for use in the dashboard interface.
